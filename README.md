CHAOS.Portal.Indexing
=====================

An extension for working with and querying Indexes on the Portal webservice hosting platform

##Requirements
.Net 4.0
Entity Framework 4.3
MySQL .Net Connector

##Code
The source code is located in the *src* folder.
